# XFMN-01 - クロスFMノイズシンセサイザー

VCV Rack用の高度なモジュレーション機能を備えたクロスFMノイズシンセサイザー

## 概要

XFMN-01は、クロスFM（周波数変調）とカオティックノイズ生成を組み合わせた実験的シンセサイザーモジュールです。オシレーター間の相互変調、有機的ノイズアルゴリズム、非線形歪み処理により、従来のFMシンセシスの境界を超えた独特な音響テクスチャを創造します。

### 主要機能

- **デュアルFMオペレーター**: 独立制御可能な2つのFMオペレーター
- **3種FMモード**: Chaos、Wide、Pureの異なるFM特性
- **ディレイエフェクト**: 高品質ディレイ処理
- **3モードフィルター**: LP/BP/HPフィルタリング
- **オクターブモード**: Sub Oct/Bypass/Upper Octの3段階
