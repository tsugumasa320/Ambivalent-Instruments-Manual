# XFMN-01 - Cross-FM Noise Synthesizer

クロスFMとノイズアルゴリズムを組み合わせた実験的なシンセサイザーモジュールです。

## 概要

XFMN-01は、クロスFM（周波数変調）とカオティックノイズ生成を統合した実験的なシンセサイザーモジュールです。オシレーター間の相互変調、有機的なノイズアルゴリズム、非線形歪み処理により、従来のFMシンセシスの境界を超えたユニークなサウンドテクスチャを創造します。

## 主な機能

- **デュアルFMオペレーター**: 独立制御可能な2つのFMオペレーター
- **3種のFMモード**: Chaos、Wide、Pure の異なるFM特性
- **ディレイエフェクト**: 高品質なディレイ処理
- **3モードフィルター**: LP/BP/HP フィルタリング
- **オクターブモード**: Sub/Bypass/Upper Oct の3段階範囲
- **包括的なCV制御**: 全パラメータのCV制御

## 信号フロー

```
[OP1] ──┬── [FM ENGINE] ──┬── [DELAY] ──┬── [FILTER] ──┬── [OUTPUT GAIN] ──┬─ [OUTPUT L/R]
         │                │
[OP2] ──┴────────────────┘
         │    [FM MODE]
```

## パラメータ

### 基本制御

- **OP1/OP2**: 2つの独立したFMオペレーター
- **FM Mode**: クロスFMノイズシンセスの特性
  - **Chaos**: カオス特性を伴う不安定な変調
  - **Wide**: 広いモジュレーション
  - **Pure**: 純粋なFMトーン

### エフェクト

- **Delay**: 高品質なディレイエフェクト
- **3-Mode Filter**: フィルタタイプ切替（LP/BP/HP）
- **Octave Mode**: サブオクターブ/バイパス/アッパーオクターブの3段階切り替え
