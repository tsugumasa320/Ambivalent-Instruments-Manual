# Modulo - Modulo Operator Utility Module

Modulo operator utility module with fixed gain scaling and dry/wet mixing.

## Features

- Modulo operation for waveform folding
- Equal-power dry/wet mixing
- Output gain control
- CV controllable parameters
- Output level indicator

## Documentation

- [English Manual](manual.md) - Complete user guide in English
- [日本語マニュアル](manual_jp.md) - 日本語での詳細ガイド

## Quick Start

1. Add Modulo to your VCV Rack patch
2. Connect audio input and output
3. Adjust Modulo parameter to control waveform folding
4. Use Mix knob to balance between dry and processed signal

For detailed instructions, see the full manual pages linked above.
