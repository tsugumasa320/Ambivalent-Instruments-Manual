# Modulo - Modulo Operator Utility Module

入力信号にモジュロ演算を適用する波形折り返しモジュールです。

## 概要

モノラル入力をモジュロ処理し、Dry/Wetとゲインで整えます。シンプルな波形加工に向いたユーティリティです。

## パラメータ

**GAIN** (0.0〜2.0、デフォルト 0.5)
- ウェット側の出力ゲイン
- **動作例**:
  - **設定値 = 0.0**: ミュート
  - **設定値 = 2.0**: 最大

**MODULO** (0.01〜1.0、デフォルト 0.5)
- モジュロ値
- **動作例**:
  - **設定値 = 0.01**: 強い折り返し
  - **設定値 = 1.0**: 折り返し少なめ

**MIX** (0.0〜1.0、デフォルト 0.5)
- ドライ/ウェット比
- **動作例**:
  - **設定値 = 0.0**: ドライのみ
  - **設定値 = 1.0**: ウェットのみ

## 入出力 / CV

- **Audio In**: Mono
- **Audio Out**: Mono
- **CV In**: Gain, Modulo, Mix (±10V)
