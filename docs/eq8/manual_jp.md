# EQ8 - 8-Band Parametric Equalizer

固定8バンドのステレオEQです。

## 概要

60Hz〜12kHzの固定バンドを持つ8バンドEQです。各バンドは±12dB、マスターは-70dB〜+6dBで調整できます。

## パラメータ

**MASTER GAIN** (-70dB〜+6dB、デフォルト 0dB)
- 全体ゲイン
- **動作例**:
  - **設定値 = -70dB**: ほぼ無音
  - **設定値 = +6dB**: 6dBブースト

**BAND1 (60Hz)** (-12dB〜+12dB、デフォルト 0dB)
- 低域ローシェルフ
- **動作例**:
  - **設定値 = -12dB**: カット
  - **設定値 = +12dB**: ブースト

**BAND2 (170Hz)** (-12dB〜+12dB、デフォルト 0dB)

**BAND3 (310Hz)** (-12dB〜+12dB、デフォルト 0dB)

**BAND4 (600Hz)** (-12dB〜+12dB、デフォルト 0dB)

## 入出力 / CV

- **Audio In**: L/R
- **Audio Out**: L/R
- **CV In**: なし

## コンテキストメニュー

- Reset All Bands
- Flat Response
- Smile Curve
- Bass Boost
- Presence Boost
- Vocal EQ
- Soft Clip Outputs (ON/OFF)
