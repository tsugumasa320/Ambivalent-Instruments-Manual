# EQ8 マニュアル

VCV Rack用の8バンド・ステレオ・パラメトリックイコライザー。音楽制作に最適化されています。

## 概要

EQ8は音楽用途に特化した8つの周波数バンドを備えた高品質イコライザーです。各バンドで±12dBの精密なゲイン制御が可能で、-70dBから+6dBのマスターゲイン機能も搭載しています。

## 主な機能

- **8固定周波数バンド**: 音楽的に最適化された周波数選択
- **ステレオ処理**: 左右チャンネル独立処理
- **高品質フィルター**: ローシェルフ/ハイシェルフとピーキングバンド（Q=1.0）
- **リアルタイム処理**: ゼロレイテンシ・オーディオ処理
- **アウトプット・ソフトクリップ**: コンテキストメニューからオプション選択
- **制作対応**: 音楽制作ワークフローに最適化

## クイックスタート

1. **オーディオ接続**: ステレオソースをLEFT/RIGHT入力に接続
2. **バンド調整**: 各周波数のスライダーでブースト/カット
3. **アウトプット**: LEFT/RIGHTアウトプットをミキサーに接続
4. **微調整**: 楽曲全体を聞きながら調整

## 周波数バンド

### 低域
- **60 Hz**: サブベース制御、基音の低域
- **170 Hz**: ベースの暖かさとパンチ
- **310 Hz**: 低中域のクリアさと輪郭

### 中域
- **600 Hz**: 中域のプレゼンスとボーカルクリアさ
- **1 kHz**: 重要な中域、ボーカル明瞭度
- **3 kHz**: 高中域、ボーカルプレゼンス

### 高域
- **6 kHz**: 高域の詳細とクリアさ
- **12 kHz**: エアー感と煌めき、倍音成分

## コントロール詳細

### ゲイン制御
- **バンド範囲**: 各バンド±12dB
- **マスター範囲**: -70dBから+6dB
- **センター位置**: 0dB（ユニティゲイン）
- **解像度**: スムーズな連続調整
- **レスポンス**: 音楽的ピーキングカーブ

### フィルター特性
- **タイプ**: 60Hzローシェルフ、12kHzハイシェルフ、6ピーキングバンド
- **Qファクター**: 1.0（音楽的帯域幅）
- **位相レスポンス**: 自然なサウンドのために最小化
- **係数計算**: リアルタイム更新

## 技術仕様

- **バンド数**: 8固定周波数バンド
- **ゲイン範囲**: 各バンド±12dB、マスター-70dBから+6dB
- **フィルタータイプ**: ローシェルフ/ハイシェルフ + ピーキング（Q=1.0）
- **処理**: リアルタイム、ゼロレイテンシ
- **チャンネル**: ステレオ（左右独立）
- **サンプルレート**: VCV Rack全サンプルレート対応

## ベストプラクティス

1. **先に削る**: 問題周波数はブーストより先にカット
2. **音楽的文脈**: 常にフルミックスの中でEQ調整
3. **穏やかな調整**: 小さな変更でより良い結果
4. **バンド選択**: 各バンドは特定の楽器要素をターゲット
5. **レベル監視**: 全体ゲイン変化に注意
