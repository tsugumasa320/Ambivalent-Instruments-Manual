# EQ8

Professional 8-band stereo parametric equalizer for VCV Rack.

## Overview

EQ8 is a precision 8-band parametric equalizer optimized for music production. It features musically tuned frequency bands with surgical gain control, making it ideal for both corrective and creative EQ tasks.
