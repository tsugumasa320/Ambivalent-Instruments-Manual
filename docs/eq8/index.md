# EQ8 - 8-Band Parametric Equalizer

EQ8 is a professional 8-band stereo parametric equalizer optimized for music production in VCV Rack.

## Features

- 8-band stereo parametric EQ
- High-precision frequency control
- Professional Q and gain parameters
- Optimized for music production workflows
- Low CPU usage with high audio quality

## Quick Start

1. Add EQ8 to your VCV Rack patch
2. Connect stereo audio input and output
3. Adjust frequency bands as needed
4. Use for mixing, mastering, or creative sound design
