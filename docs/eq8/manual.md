# EQ8 - 8-Band Parametric Equalizer

Stereo EQ with 8 fixed frequency bands.

## Overview

Fixed bands from 60Hz to 12kHz. Each band offers Â±12dB gain, with master gain from -70dB to +6dB.

## Parameters

**MASTER GAIN** (-70dB to +6dB, default 0dB)
- Overall gain
- **Examples**:
  - **Value = -70dB**: Near silent
  - **Value = +6dB**: 6dB boost

**BAND1 (60Hz)** (-12dB to +12dB, default 0dB)
- Low shelf
- **Examples**:
  - **Value = -12dB**: Cut
  - **Value = +12dB**: Boost

**BAND2 (170Hz)** (-12dB to +12dB, default 0dB)
- **Examples**:
  - **Value = -12dB**: Cut
  - **Value = +12dB**: Boost

**BAND3 (310Hz)** (-12dB to +12dB, default 0dB)
- **Examples**:
  - **Value = -12dB**: Cut
  - **Value = +12dB**: Boost

**BAND4 (600Hz)** (-12dB to +12dB, default 0dB)
- **Examples**:
  - **Value = -12dB**: Cut
  - **Value = +12dB**: Boost

**BAND5 (1kHz)** (-12dB to +12dB, default 0dB)
- **Examples**:
  - **Value = -12dB**: Cut
  - **Value = +12dB**: Boost

**BAND6 (3kHz)** (-12dB to +12dB, default 0dB)
- **Examples**:
  - **Value = -12dB**: Cut
  - **Value = +12dB**: Boost

**BAND7 (6kHz)** (-12dB to +12dB, default 0dB)
- **Examples**:
  - **Value = -12dB**: Cut
  - **Value = +12dB**: Boost

**BAND8 (12kHz)** (-12dB to +12dB, default 0dB)
- High shelf
- **Examples**:
  - **Value = -12dB**: Cut
  - **Value = +12dB**: Boost

## I/O / CV

- **Audio In**: L/R
