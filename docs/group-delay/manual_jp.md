# GroupDelay - All-Pass Disperser

オールパスフィルタ群による位相分散エフェクトです。

## 概要

周波数ごとの位相遅延を作ることでトランジェントや空間感を変化させます。FREQ/AMOUNT/PINCHで特性を調整します。キックなどのアタックを強調したい素材で使うと分かりやすいです。Kilohearts Disperser にインスパイアされています。

## パラメータ

**FREQ** (0.0〜1.0、デフォルト 0.5)
- 中心周波数（表示は20Hz〜20kHz）
- **動作例**:
  - **設定値 = 0.0**: 20Hz付近
  - **設定値 = 1.0**: 20kHz付近

**FREQ CV DEPTH** (-1.0〜1.0、デフォルト 0.0)
- FREQ CVの深さ
- **動作例**:
  - **設定値 = 0.0**: CV無効
  - **設定値 = 1.0**: 最大深さ

**AMOUNT** (0.0〜32.0、デフォルト 4.0)
- フィルター段数
- **動作例**:
  - **設定値 = 0**: ほぼ無効
  - **設定値 = 32**: 最大効果

**AMOUNT CV DEPTH** (-1.0〜1.0、デフォルト 0.0)
- AMOUNT CVの深さ
- **動作例**:
  - **設定値 = 0.0**: CV無効
  - **設定値 = 1.0**: 最大深さ

**PINCH** (0.5〜16.0、デフォルト 5.15)
- Q値
- **動作例**:
  - **設定値 = 0.5**: 広いピーク
  - **設定値 = 16.0**: 鋭いピーク

**PINCH CV DEPTH** (-1.0〜1.0、デフォルト 0.0)
- PINCH CVの深さ
- **動作例**:
  - **設定値 = 0.0**: CV無効
  - **設定値 = 1.0**: 最大深さ

## 入出力 / CV

- **Audio In**: L/R
- **Audio Out**: L/R
- **CV In**: Frequency, Amount, Pinch (±10V)

## コンテキストメニュー

- Freq CV Filter（CV平滑化のON/OFF）
