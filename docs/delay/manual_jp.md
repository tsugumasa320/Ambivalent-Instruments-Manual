# Delay - ステレオ・ディレイ・モジュール

Repitch / Octave / Reverse / Ping-Pong を切り替えられるステレオ・ディレイです。

## 概要

0.001〜5.0秒のディレイタイム、フィードバック、トーン、モジュレーションを備えたステレオ・ディレイです。MODEボタンで4種類のディレイ挙動を切り替えます。

## パラメータ

**TIME** (0.0〜1.0、デフォルト 0.5)
- ディレイ時間（表示は0.001〜5.0秒）
- **動作例**:
  - **設定値 = 0.0**: 0.001秒
  - **設定値 = 1.0**: 5.0秒

**FEEDBACK** (0.0〜0.95、デフォルト 0.5)
- フィードバック量
- **動作例**:
  - **設定値 = 0.0**: 反復なし
  - **設定値 = 0.9**: 長い反復

**WET** (0.0〜1.0、デフォルト 0.5)
- ドライ/ウェット比
- **動作例**:
  - **設定値 = 0.0**: ドライのみ
  - **設定値 = 1.0**: ウェットのみ

**TONE** (0.0〜1.0、デフォルト 0.5)
- フィードバックのトーン（0.0=暗い、1.0=明るい）
- **動作例**:
  - **設定値 = 0.0**: ローパス寄り
  - **設定値 = 1.0**: ハイパス寄り

**MOD** (0.0〜1.0、デフォルト 0.0)
- モジュレーション深度
- **動作例**:
  - **設定値 = 0.0**: モジュレーションなし
  - **設定値 = 1.0**: 強い揺らぎ

**MODE** (ボタン、デフォルト Repitch)
- 押すたびに Repitch / Octave / Reverse / Ping-Pong を循環

## 入出力 / CV

- **Audio In**: L/R
- **Audio Out**: L/R
- **CV In**: Time, Tone, Feedback, Wet (±10V)

## コンテキストメニュー

- 追加項目なし
