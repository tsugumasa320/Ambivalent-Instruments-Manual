# Grainer - Granular Synthesis Processor

**注意：このマニュアルはAI生成によって作られており、順次更新していく予定です。**

リアルタイム・グラニュラーシスプロセッサーです。

## 概要

入力信号をリアルタイムにグラニュラライズし、ライブ演奏用に音響テクスチャを作成できます。最大128グレイン（デフォルト32）と5秒のレコーディングバッファを持ち、精確なコントロールと創造的な予測性を両立しています。

## クイックスタート

1. **オーディオ接続**: AUDIO INPUT L/R に音源を接続
2. **GRAIN LENGTH 調整**: GRAIN LENGTH ノブでグレイン長を調整（50ms - 1000ms、デフォルト525ms）
3. **DENSITY 調整**: DENSITY ノブで密度を設定（表示 0 - 50Hz）
4. **音色調整**: PITCH / POSITION / STEREO SPREAD でサウンドをシェイプ

## パラメータ

### 基本制御

- **GRAIN LENGTH**: 個別グレイン長（50ms - 1000ms、デフォルト525ms）
- **DENSITY**: グレイン生成レート（0 - 50Hz 表示）
- **PITCH**: ピッチシフト比（1/3x - 3x、中央 = 1x）
- **POSITION**: 再生位置（0 - 100%: 0% = 最古、100% = REC位置）
- **STEREO SPREAD**: ランダムなステレオ配置（0 - 1）
  - **0%**: モノ/センター配置（全グレインがセンターにパン）
  - **50%**: 中程度のステレオスプレッド（センター周り±25%パン範囲）
  - **100%**: 完全なステレオランダマイズ（グレインがステレオフィールドのどこにでも配置）

### 高度制御

- **PITCH RANDOM**: ピッチランダム量（0 - 3）
  - **0**: ピッチランダムなし（全グレインが正確なピッチ設定を使用）
  - **1**: ±1オクターブランダム変化（2^-1 〜 2^+1 = 0.5x 〜 2.0x）
  - **2**: ±2オクターブランダム変化（2^-2 〜 2^+2 = 0.25x 〜 4.0x）
  - **3**: ±3オクターブランダム変化（2^-3 〜 2^+3 = 0.125x 〜 8.0x）
- **PITCH QUANTIZE**: ピッチを特定のピッチ比に量子化する度合いを制御（0 - 1、デフォルト1.0、ダブルクリックで1.0にリセット）
  - **LED表示**: 音楽的インターバルに近い明るさで色分けされた輝度を表示
  - **動作例**:
    - **QUANTIZE = 0**: ピッチは連続的に変化（量子化なし）
    - **QUANTIZE = 0.5**: ピッチは0.5x, 1.0x, 1.5x, 2.0x... などの倍数にスナップする
    - **QUANTIZE = 1.0**: 完全に整数比（1.0x, 2.0x, 3.0x...）に量子化
  - **LEDの明るさ**: 有理数計算によりシンプルな音程比（オクターブ、完全五度など）に近いほど明るく表示されます
    - **暗いLED**: ピッチ値が音楽的比の間にある
    - **消灯**: 明確な音楽的比にマッチしない

### ウィンドウ形状

- **WINDOW SHAPE**: ハン窓形状のバイアス（-1 〜 +1）
  - **-1**: 左バイアスされた窓形状
  - **0**: 対称ハン窓（センター位置）
  - **+1**: 右バイアスされた窓形状
- **POSITION RANDOM**: 位置のランダム変化（0 - 1）

### 入力ゲイン

- **INPUT GAIN**: 入力ゲイン（0 - 1 リニア）
  - **0%**: 入力なし
  - **100%**: フル入力ゲイン

### ドライ/ウェット

- **DRY/WET**: 原音と処理音のバランス（0 - 1）
  - **0%**: 100% ド音（元の信号のみ）
  - **100%**: 100% ウェット（処理音のみ）
  - **中間値**: ド音と処理音のブレンド

### リバーブ

- **REVERB**: 内蔵リバーブ量（0 - 1）
  - **0%**: リバーブなし
  - **100%**: 最大リバーブエフェクト（Mutable Instruments Cloudsベース）

### フォワード/リバース

- **FORWARD/REVERSE**: 再生方向切替
  - **FORWARD**: 通常の再生方向（デフォルト）
- **REVERSE**: 逆再生方向

### フリーズ

- **FREEZE**: バッファ内容を固定（ボタン/CVゲート）
- **OFF**: 通常録音モード（バッファが継続的に更新されます）
- **ON**: バッファ内容がフリーズされます（新しい録音なし、既存の内容を保持）
