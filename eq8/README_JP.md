# EQ8 - 8バンド・パラメトリックEQ

VCV Rack向けの8バンド・ステレオ・パラメトリックイコライザーです。

## 概要

EQ8は音楽制作に最適化した8つの固定周波数バンドを搭載した高品質EQです。各バンドは±12dBのゲイン調整が可能で、マスターゲインは -70dB〜+6dB の範囲です。音楽的な帯域幅（Q=1.0）のバイクアッドフィルターで処理します。

## 主な機能

- **8固定周波数バンド**: 音楽用途に最適化した選定
- **ステレオ処理**: L/R独立処理
- **高品質フィルター**: ロー/ハイ・シェルフ + ピーキング (Q=1.0)
- **リアルタイム処理**: ゼロレイテンシ
- **ソフトクリップ**: コンテキストメニューから出力ソフトクリップを有効化
- **制作向け設計**: ミックス／マスタリングを想定

## クイックスタート

1. **入力**: LEFT/RIGHT にステレオ入力
2. **調整**: スライダーで欲しい帯域をブースト/カット
3. **出力**: LEFT/RIGHT 出力をミキサーへ
4. **微調整**: 楽曲全体の文脈で調整

## 周波数バンド

### 低域
- **60 Hz**: サブベース／低域の芯
- **170 Hz**: ベースの厚みとパンチ
- **310 Hz**: ローミッドの明瞭さ

### 中域
- **600 Hz**: 中域の存在感
- **1 kHz**: ボーカルの明瞭さ
- **3 kHz**: プレゼンス

### 高域
- **6 kHz**: ディテールとアタック
- **12 kHz**: 空気感と輝き

## コントロール詳細

### ゲイン
- **バンド**: ±12dB
- **マスター**: -70dB〜+6dB
- **中央**: 0dB
- **連続可変**: スムーズな調整

### フィルター特性
- **タイプ**: 60Hzローシェルフ、12kHzハイシェルフ、6バンドピーキング
- **Q**: 1.0
- **処理**: リアルタイム係数更新

## 応用例

- 収録音の補正EQ
- ライブ時のトーン調整
