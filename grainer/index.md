# Grainer - Granular Synthesis Processor

Grainer is a real-time granular synthesis processor for VCV Rack that transforms audio through advanced grain-based processing.

## Features

- Real-time granular synthesis with multi-grain engine
- Advanced modulation capabilities  
- High-quality audio processing optimized for music production
- Professional-grade parameter controls

1. Add Grainer to your VCV Rack patch
2. Connect audio input and output
3. Adjust grain parameters to taste
4. Use CV inputs for dynamic modulation
